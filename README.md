LCMS1.0
=======

##Lone Content Management System V 1.0 安装

解压文件后上传所有文件到服务器根目录。

将LCMS目录修改为你想要的名称，以防他人非法利用。

在浏览器输入： http://你的域名/LCMS目录/Install/

按提示操作。

##例如：
你的网站域名是www.zhuiyun.net, LCMS目录修改为MyCMS
则安装时的路径为：http://www.zhuiyun.net/MyCMS/Install/

安装完成后，程序自动删除Install目录，以防止重复安装出错。

如需重新安装，请删除原有数据库表，拷贝Install目录到LCMS下运行即可。

##注意：
LCMS系统要求服务器有FSO写入权限，否则无法正常安装。
